## Executive Summary

This document outlines the **compact keyword patterns** for SmartFirm.io's programmatic SEO buildout. Following Edward Sturm's methodology, we're targeting **hyper-specific, bottom-of-funnel (BOFU) keywords** that signal strong purchase intent. Each landing page will be **400-500 words** with a direct conversion focus, rather than long-form blog content.

**Key Principle:** Target users who know what they want and are ready to hire, not those in the research phase.

---

## What Makes a "Compact Keyword"?

**Compact Keywords are:**
- **Specific:** Include modifiers that signal exact needs (e.g., "with automated follow-up")
- **High-Intent:** Indicate readiness to buy/hire (bottom-of-funnel)
- **Low-Competition:** Hyper-specific combinations that competitors overlook
- **Conversion-Ready:** Lead directly to booking a call or signing up

**Examples:**
- ✅ **Compact:** "CPA marketing with automated lead follow-up"
- ❌ **Non-Compact:** "CPA marketing services" (too broad, competitive)
- ❌ **Non-Compact:** "What is CPA marketing?" (top-of-funnel, informational)

---

// pages/services/technical/[slug].js
export async function getStaticPaths() {
  const pages = getProgrammaticPages()

  // Only generate paths for published pages
  const publishedPages = pages.filter(page => {
    return page.published || new Date(page.publishDate) <= new Date()
  })

  const paths = publishedPages.map(page => ({
    params: { slug: page.slug }
  }))

  return { paths, fallback: 'blocking' }
}

export async function getStaticProps({ params }) {
  const page = getPageBySlug(params.slug)

  // Return 404 if page not published yet
  if (!page.published && new Date(page.publishDate) > new Date()) {
    return { notFound: true }
  }

  return { props: { page } }
}

## Sitemap Update Strategy

### **Option 1: Dynamic Sitemap (Recommended)**
Generate sitemap dynamically based on published pages only

```javascript
// pages/sitemap.xml.js
export async function getServerSideProps({ res }) {
  const pages = getProgrammaticPages()
  const publishedPages = pages.filter(p => p.published)

  const sitemap = `<?xml version="1.0" encoding="UTF-8"?>
    <urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
      ${publishedPages.map(page => `
        <url>
          <loc>https://smartfirm.io/services/technical/${page.slug}</loc>
          <lastmod>${page.publishDate}</lastmod>
          <changefreq>monthly</changefreq>
          <priority>0.8</priority>
        </url>
      `).join('')}
    </urlset>
  `

  res.setHeader('Content-Type', 'text/xml')
  res.write(sitemap)
  res.end()

  return { props: {} }
}
```
